RUNGUN


About this project

RunGun started as a way to kill time in an especially boring end-of-year segment of a high school Computer Science class. Inspired by its predecessor, WalkPunch, RunGun was born. RunGun was made with the full permission of the original creators of WalkPunch, who support this game in its entirety.

For a time, this project was moved into Java as a learning exercise for that language. After determining that it would be more accessible in its native HTML/JS format, it was ported back. Ever since, the project has been occasionally updated and still receives sporadic updates. Typos are also fixed when found.

FAQ
