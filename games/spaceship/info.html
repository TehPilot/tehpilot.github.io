Heads up: The game project is in an early development state. Glitches, unexpected behavior, and incomplete features may appear in some of these builds. spaceship game is tested on Windows 7 and 10 systems, and will only work on Windows systems currently. Presentations of the game use four Xbox 360 controllers.

Any Windows-compatible controller (without use of an external adapter) that has all of the buttons below should work with the game.

Controls:
A - Select
B - Cancel
Left Stick - Move
Right Stick - Fire
Left/Right Trigger - Special Abilities
Left/Right Bumper - Alternate Movement/Fire Options
Select / Back - Exit Results Screen
